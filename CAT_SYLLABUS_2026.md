# CAT Syllabus 2026 Cross-Check

This is the readable reference file for the syllabus screenshots.

The app tracker data is in `lib/cat_data.dart` under `catTopics` and
`topicSubtopics`. Use this file to review, add, delete, or rename syllabus
items first, then mirror the change in `lib/cat_data.dart` if you want it to
show inside the app.

## VARC

### Reading Comprehension

Pattern from screenshot:

- 16 questions
- 4 questions each from 4 RCs
- No fixed RC syllabus, but recurring passage themes are common

RC passage themes:

- Philosophy
- Psychology
- History
- Arts and museum
- Society
- Culture
- Biology
- Science and technology

App tracker topics:

- Main idea and central theme
- Inference and implication
- Tone, stance, and purpose
- Detail, title, and structure questions
- RC passage themes

### Verbal Ability

Pattern from screenshot:

- 8 questions
- Usually 2 questions each from the main VA formats

Topics:

- Para summary
- Para jumbles
- Para completion / fill in the blank
- Odd one out / out of context

App tracker topics:

- Para summary
- Para completion, insertion, and fill in the blank
- Odd sentence out / out of context
- Para jumbles
- Critical reasoning basics
- Grammar and vocabulary by context

## DILR

### Logical Reasoning

Pattern from screenshot:

- Out of 5 sets, at least 2-3 sets can be logical reasoning based

Topics:

- Puzzles
- Arrangements
- Selection
- Games and tournaments
- Conditional LRs
- Coin picking
- Binary logic
- Truth and lie
- Case-based LRs
- 2D and 3D LR

App tracker topics:

- Linear, circular, and grid arrangements
- Selection and distribution
- Scheduling, order, and ranking
- Games and tournaments
- Routes, networks, and paths
- Binary logic and truth conditions
- Puzzles and case-based LR
- Hybrid logic-data sets

### Data Interpretation

Pattern from screenshot:

- Around 1-2 sets of data interpretation questions in each slot
- Focuses on analyzing and interpreting data in tables, charts, and graphs

Topics:

- Tables based DI
- Caselets
- Quant based DI
- Graphs
- Charts
- Networks and diagrams
- Routes and networks
- Mathematical puzzle
- Venn diagram
- Maxima-minima

App tracker topics:

- Tables, graphs, and chart sets
- Caselets and multi-source data
- Percent, ratio, average data logic
- Quant, network, and mathematical DI
- Routes, networks, and paths
- Hybrid logic-data sets
